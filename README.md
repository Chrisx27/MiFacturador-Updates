MiFacturador es un sistema de escritorio para la gestión integral de facturas y documentos PDF, diseñado especialmente para pequeños negocios, autónomos y emprendedores que buscan una solución sencilla pero potente.

Características principales:
📄 Gestión completa de facturas: Crear, editar, eliminar y visualizar facturas con campos personalizables (NIF/CIF, teléfono, país, etc.).

🔍 Lectura automática de PDFs: Extrae fecha, número de factura y monto total desde archivos PDF, incluso si son escaneados (gracias a OCR integrado con EasyOCR).

📂 Organización inteligente: Los PDFs se guardan automáticamente en carpetas estructuradas por año, mes y cliente.

📊 Reportes mensuales en Excel: Genera informes detallados con totales por cliente, filtros y campos dinámicos.

⏱️ Control de pagos diferidos (pospago): Gestiona facturas con fecha de vencimiento y recibe alertas.

👥 Autocompletado inteligente: Al escribir el nombre de un cliente, sugiere coincidencias y rellena automáticamente los campos con los datos de su última factura.

🔄 Actualizaciones automáticas: El botón "Buscar actualizaciones" descarga e instala nuevas versiones directamente desde GitHub.

🖥️ Instalador profesional: Incluye un instalador para Windows que crea accesos directos y permite desinstalación limpia, sin necesidad de conocimientos técnicos.

Tecnologías utilizadas:
Python con CustomTkinter (interfaz moderna)

Base de datos SQLite (ligera y portable)

PyMuPDF + EasyOCR para lectura de PDFs

PyInstaller + Inno Setup para distribución

Ideal para llevar un control ordenado y eficiente de tus facturas, con una interfaz moderna, oscura y fácil de usar. ¡Totalmente funcional y listo para usar!

¿Por qué usar MiFacturador? Porque combina la potencia de un software profesional con la sencillez de una herramienta hecha a medida, sin costos de licencia ni complicaciones.
